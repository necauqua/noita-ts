# @noita-ts/noita-dear-imgui-types

TypeScript type definitions for
[Noita-Dear-ImGui](https://github.com/dextercd/Noita-Dear-ImGui) (ImGui version
1.26.0), for use with [noita-ts](https://github.com/necauqua/noita-ts) /
[TypeScriptToLua](https://typescripttolua.github.io/).

These definitions were generated with an LLM-written Python script that reuses
Noita-Dear-ImGui's existing LuaLS docgen (also written in Python) to emit
TypeScript typings instead of LuaLS annotations.
